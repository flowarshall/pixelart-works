2. UI Reference

🔗 NHN Careers (https://careers.nhn.com/)
  • 기존 백화점식 대형 편입학원의 복잡하고 답답한 레이아웃을 탈피하고, 시원한 여백과 타이포그래피를 강조한 와이드 레이아웃을 제안드립니다.
  • 역동적인 블루 톤에 학생들의 열정이 돋보이는 따뜻한 감성 이미지를 적재적소에 매치하여 진취적인 무드를 연출합니다.
  • 사용자의 시선을 짧은 영문에 담은 핵심 카피와 전환 요소에 온전히 집중시켜, BLIS 편입만의 1% 프리미엄 교육 철학과 '초신성(Rising Star)'으로서의 위상을 직관적으로 전달합니다.
🔗 MEGAZONE (https://www.megazone.com/)
  • 명확한 블루/블랙 컬러 대비와 모서리가 둥근(Round) 카드 UI를 활용하여, 세련되고 트렌디한 플랫폼 이미지를 구축할 것을 제안드립니다.
  • 가로형 띠 배너와 부드러운 그라데이션, 솔리드 컬러를 조화롭게 사용하여 시각적 지루함을 없애고 핵심 콘텐츠로의 주목도를 극대화합니다.

편입의 판도를 바꾼 라이징 스타, BLIS 편입의 압도적 1% 클래스.

기획 의도 기존 업계와 대비되는 '젊고 강력한 신흥 강자'의 이미지를 구축하여, 2026년 연고대 89명 최종합격자를 배출하며 폭발적으로 성장한 BLIS 편입의 압도적인 실적 데이터를 첫 화면에서 직관적으로 각인시킵니다.
기대 효과 사이트 진입 직후 프리미엄 포지셔닝을 선점하여, 대형 학원 대비 높은 수강료에 대한 고객의 심리적 저항을 차단하고 '연고대 편입 전문'이라는 압도적인 신뢰를 부여합니다.

BLIS만의 전문 편입설명회 영상

연고대·인서울 TOP7 편입 전문가의 컨설팅을 온라인으로 지금 바로 경험해 보세요.

기획 의도 타 편입 학원에서는 제공하지 않는 온라인 편입설명회 영상을 제공하여 대체 불가능한 프리미엄 학원임을 증명하고, 메인페이지 내에서 간편히 접근 가능하도록 세팅합니다.
기대 효과 오프라인 설명회보다 접근성이 높은 온라인 영상 제공으로 사용자의 정보 탐색 니즈를 직접 충족시키며, 편입 시장 내 최고의 전문가 집단으로서의 권위를 확고히 다집니다.
